How to install

After downloading .zip with all the elements, follow the following steps.


  1. Unzip the .zip files in the folder you want to design the web
  2. The index.html which provides an example file to understand the main operating framework
  3. Links to styles and javascript files to be created are:

<link rel="stylesheet" type="text/css" href="css/index.css"/>

<link rel="stylesheet" type="text/css" href="font-awesome-4.2.0/css/font-awesome.min.css"/>

<script src="js/jquery-2.1.1.js"></script>
